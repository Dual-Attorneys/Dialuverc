# Dialuverc

A narrative framework for games, written with [Dual Attorneys](discord.gg/phECHVHCDe) in mind.

## Status

This project is **very** early in its development, and as such, is constantly changing.

Examples of its current usage can be found in [DualAttorneys.Dialuverc](https://github.com/Dual-Attorneys/Dialuverc/tree/main/DualAttorneys.Dialuverc).

## Objective

*Dialuverc* intends to be a one-stop-shop framework to be used with any C# project, attempting to best accommodate both writers and programmers.
This means providing some powerful and extensible tools out-of-the-box for the former, while maintaining type safety wherever possible for the latter.

In addition to runtime story execution, it provides a base to build your own editor, allowing you to design stories and related custom mechanics as you see fit.

## Requirements

.NET 8.0
